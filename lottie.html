Lottie 動畫套件

Lottie 是一個強大的動畫套件，可以讓您在網頁中輕鬆播放 After Effects 動畫。訪問 官方網站 了解更多。

特色介紹
  • 🎨 支援複雜的向量動畫
  • 📱 跨平台支援
  • ⚡ 輕量化檔案大小
  • 🛠️ 豐富的控制 API
使用方式

1. 引入 Lottie 函式庫

2. 準備 JSON 動畫檔

3. 建立容器元素

4. 載入並控制動畫

程式碼範例


// HTML 部分：建立一個容器來放置動畫
<div id="lottie-container" style="width: 400px; height: 400px;"></div>

// JavaScript 部分：
// 1. 初始化動畫
const animation = lottie.loadAnimation({
  container: document.getElementById('lottie-container'), // 指定容器元素
  renderer: 'svg',    // 渲染方式，可選 'svg' 或 'canvas'
  loop: true,        // 是否循環播放
  autoplay: true,    // 是否自動播放
  path: 'animation.json'  // 動畫檔案路徑
});

// 2. 動畫控制方法
animation.play();      // 開始播放動畫
animation.pause();     // 暫停在當前幀
animation.stop();      // 停止並回到第一幀
animation.setSpeed(2); // 設定播放速度(2倍速)

// 3. 動畫事件監聽
animation.addEventListener('complete', () => {  // 動畫播放完成時
  console.log('動畫播放完成');
});

animation.addEventListener('loopComplete', () => {  // 每次循環完成時
  console.log('完成一次循環');
});

// 4. 進階控制
animation.goToAndPlay(30, true);   // 跳轉到第30幀並播放
animation.goToAndStop(50, true);   // 跳轉到第50幀並停止
animation.setDirection(-1);        // 設定播放方向(1正向、-1反向)
animation.getDuration(true);       // 獲取動畫總時長(秒)

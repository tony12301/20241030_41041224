Chart.js 圖表動畫套件

Chart.js 是一個簡單且靈活的 JavaScript 圖表庫，能夠建立互動式的動態圖表。訪問 官方網站 了解更多。

特色介紹
  • 📊 8種圖表類型
  • 🎨 高度可自訂化
  • 📱 響應式設計
  • ✨ 流暢動畫效果
使用方式

1. 引入 Chart.js 函式庫

2. 準備 canvas 元素

3. 設定圖表數據

4. 初始化圖表

程式碼範例


// 取得 canvas 元素
const ctx = document.getElementById('myChart').getContext('2d');

// 建立圖表
new Chart(ctx, {
  type: 'bar',           // 圖表類型
  data: {
    labels: ['一月', '二月', '三月'],
    datasets: [{
      label: '銷售量',
      data: [12, 19, 3],
      backgroundColor: [
        'rgba(255, 99, 132, 0.2)',
        'rgba(54, 162, 235, 0.2)',
        'rgba(255, 206, 86, 0.2)'
      ],
      borderColor: [
        'rgba(255, 99, 132, 1)',
        'rgba(54, 162, 235, 1)',
        'rgba(255, 206, 86, 1)'
      ],
      borderWidth: 1
    }]
  },
  options: {
    responsive: true,    // 響應式
    animation: {         // 動畫設定
      duration: 2000,    // 持續時間
      easing: 'easeOutQuart'
    }
  }
});

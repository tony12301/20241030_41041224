文字轉語音套件

Web Speech API 是一個能將文字轉換成語音的 JavaScript API，讓網頁具有朗讀功能。訪問 官方網站 了解更多。

特色介紹
  • 🗣️ 支援多國語言
  • ⚙️ 可調整語音速度和音調
  • 📱 跨瀏覽器支援
  • 🔊 即時語音合成
使用方式

1. 建立 SpeechSynthesis 實例

2. 設定語音參數

3. 選擇語音語言

4. 呼叫 speak() 方法

程式碼範例


// 建立語音合成實例
const synth = window.speechSynthesis;

// 建立語音設定
const utterance = new SpeechSynthesisUtterance('要朗讀的文字');

// 基本設定
utterance.lang = 'zh-TW';     // 設定語言為中文
utterance.rate = 1.0;         // 設定語速 (0.1-10)
utterance.pitch = 1.0;        // 設定音調 (0-2)
utterance.volume = 1.0;       // 設定音量 (0-1)

// 事件處理
utterance.onstart = () => {   // 開始朗讀時
  console.log('開始朗讀');
}
utterance.onend = () => {     // 結束朗讀時
  console.log('朗讀結束');
}

// 開始朗讀
synth.speak(utterance);       // 執行朗讀

// 停止朗讀
// synth.cancel();           // 取消朗讀

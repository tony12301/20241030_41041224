Anime.js 動畫套件

Anime.js 是一個強大的 JavaScript 動畫庫，可以輕鬆創建複雜的動畫效果。訪問 官方網站 了解更多。

特色介紹
  • ✨ 輕量級且高效能
  • 🎨 支援 CSS、SVG、DOM 屬性動畫
  • 📱 豐富的緩動函數
  • ⚡ 時間軸控制功能
使用方式

1. 引入 Anime.js 函式庫

2. 選擇目標元素

3. 設定動畫參數

4. 執行動畫

程式碼範例


// 創建一個新的動畫實例
anime({
  // 設定動畫目標元素 (使用 CSS 選擇器)
  targets: '.element',
  
  // 水平移動 250 像素
  translateX: 250,
  
  // 旋轉一圈 (360度)
  rotate: '1turn',
  
  // 動畫持續時間為 800 毫秒
  duration: 800,
  
  // 使用 easeInOutQuad 緩動函數
  // 開始和結束時較慢,中間較快
  easing: 'easeInOutQuad'
});

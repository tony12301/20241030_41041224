Fabric.js 互動式畫布套件

Fabric.js 是一個強大的 JavaScript 畫布處理函式庫，能讓你輕鬆創建和操作互動式的圖形物件。訪問 官方網站 了解更多。

特色介紹
  • 🎨 豐富的繪圖功能
  • 🖱️ 互動式物件操作
  • 📱 支援觸控設備
  • 🔄 SVG 轉換支援
使用方式

1. 引入 Fabric.js 函式庫

2. 建立 Canvas 元素

3. 初始化 Fabric 畫布

4. 新增和操作物件

程式碼範例


// 初始化 Fabric 畫布
var canvas = new fabric.Canvas('canvas');

// 創建矩形
var rect = new fabric.Rect({
  left: 100,
  top: 100,
  fill: 'red',
  width: 50,
  height: 50
});

// 創建圓形
var circle = new fabric.Circle({
  radius: 30,
  fill: 'blue',
  left: 200,
  top: 100
});

// 將物件加入畫布
canvas.add(rect);
canvas.add(circle);

// 監聽物件修改事件
canvas.on('object:modified', function(e) {
  console.log('物件已被修改');
});

FullCalendar 日曆套件

FullCalendar 是一個功能豐富的 JavaScript 日曆套件，可以幫助您輕鬆建立互動式日曆和事件管理系統。訪問 官方網站 了解更多。

特色介紹
  • 📅 支援多種視圖模式（月、週、日）
  • 🔄 可拖放事件
  • 🌐 支援多國語言
  • 📱 響應式設計
  • 🎨 高度可自訂化
主要功能

1. 事件管理與顯示

2. 日期範圍選擇

3. 事件拖放功能

4. 多種視圖切換

程式碼範例


// 初始化日曆
document.addEventListener('DOMContentLoaded', function() {
  var calendarEl = document.getElementById('calendar');
  var calendar = new FullCalendar.Calendar(calendarEl, {
    initialView: 'dayGridMonth',
    locale: 'zh-tw',
    headerToolbar: {
      left: 'prev,next today',
      center: 'title',
      right: 'dayGridMonth,timeGridWeek,timeGridDay'
    },
    events: [
      {
        title: '示範事件1',
        start: '2024-11-11'
      }
    ]
  });
  calendar.render();
});

互動示範，可點擊日期新增事件，點擊事件刪除事件

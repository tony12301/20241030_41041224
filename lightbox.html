Lightbox2 圖片展示套件

Lightbox2 是一個輕量級的 JavaScript 圖片展示套件，可以讓網頁中的圖片以燈箱效果展示。訪問 官方網站 了解更多。

特色介紹
  • 🖼️ 優雅的圖片展示效果
  • ⚡ 輕量級且易於使用
  • 📱 支援響應式設計
  • 🎯 支援圖片群組展示
使用方式

1. 引入 Lightbox2 CSS 和 JS

2. 設定圖片連結和屬性

3. 加入 data-lightbox 屬性

4. 可選擇性加入圖片說明

程式碼範例


// 單張圖片展示
// href: 指向大圖的路徑
// data-lightbox: 用於啟動 lightbox 效果，單張圖片可使用唯一值
// data-title: 顯示在燈箱中的圖片說明文字
<a href="large-image.jpg" data-lightbox="image-1" data-title="圖片說明">
    // src: 縮圖路徑，點擊後會顯示 href 中的大圖
    // alt: 圖片的替代文字，對無障礙功能很重要
    <img src="thumbnail.jpg" alt="縮圖">
</a>

// 群組展示 - 多張圖片使用相同的 data-lightbox 值可以組成相冊
// 使用者可以在燈箱中前後切換瀏覽群組中的圖片
<a href="image1.jpg" 
   data-lightbox="gallery"  // 相同的 gallery 值將圖片分組
   data-title="圖片1"       // 每張圖片可以有獨立的說明文字
>
    <img src="thumb1.jpg" alt="第一張圖片縮圖">
</a>

<a href="image2.jpg" 
   data-lightbox="gallery"  // 使用相同的 gallery 值
   data-title="圖片2"       // 燈箱展示時會顯示的說明文字
>
    <img src="thumb2.jpg" alt="第二張圖片縮圖">
</a>

P5.js 創意程式設計套件

P5.js 是一個用於創意編程和互動視覺設計的 JavaScript 函式庫，讓你輕鬆創建數位藝術作品。訪問 官方網站 了解更多。

特色介紹
  • 🎨 豐富的繪圖功能
  • 🎵 音效與互動支援
  • 📱 跨平台相容性
  • 🔄 動畫與物理模擬
使用方式

1. 引入 P5.js 函式庫

2. 設定 setup() 函數

3. 實作 draw() 函數

4. 加入互動功能

程式碼範例


// 設定畫布大小和初始化
function setup() {
  // 建立 400x400 像素的畫布
  createCanvas(400, 400);
  // 設定背景色為白色
  background(255);
  // 設定描邊顏色為黑色
  stroke(0);
  // 設定填充顏色為淺灰色
  fill(200);
}

// 繪製循環
function draw() {
  // 每幀重新繪製背景,設為淺灰色
  background(220);
  
  // 繪製跟隨滑鼠的圓形
  // mouseX, mouseY 為滑鼠座標
  // 50, 50 為圓形的寬高
  ellipse(mouseX, mouseY, 50, 50);
  
  // 在圓形中心加入十字線
  line(mouseX - 10, mouseY, mouseX + 10, mouseY);
  line(mouseX, mouseY - 10, mouseX, mouseY + 10);
}
